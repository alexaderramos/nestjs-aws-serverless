## Description

Prueba tecnica Softek

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Deploy

```bash
# install dependencies
$ npm i -g serverless

# compile
$ npm run build

# deploy lambda
$ serverless deploy
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
